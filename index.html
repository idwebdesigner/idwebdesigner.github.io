Sebelum mengubah lokasi...

Pastikan anda sudah mengaktifkan mode pengembang dan memilih Zimbabwe di bagian aplikasi lokasi palsu

Pastikan langkah berikut sudah dilakukan:

  1. Buka Pengaturan > Tentang Ponsel.
  2. Ketuk Build Number (Nomor Versi) sebanyak 7x hingga muncul pesan "Mode Pengembang Aktif".
  3. Buka Opsi Pengembang (Developer Options).
  4. Cari menu "Pilih aplikasi lokasi palsu" (Select mock location app).
  5. Pilih aplikasi GPS Zimbabwe
