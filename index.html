Make sure you have enabled Developer Mode and selected Zimbabwe in the Mock location app section.

Pastikan anda sudah mengaktifkan mode pengembang dan memilih Zimbabwe di bagian aplikasi lokasi palsu.

  1. Open Setting (Pengaturan) > About Phone (Tentang Ponsel).
  2. Tap Build Number (Nomor Versi) 7x hingga muncul pesan Developer Mode Active (Mode Pengembang Aktif).
  3. Open Developer Options (Opsi Pengembang).
  4. Find "Select mock location app" (Pilih aplikasi lokasi palsu).
  5. Pilih Zimbabwe GPS Dev
